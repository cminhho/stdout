# Keyboard Shortcuts

Global shortcuts let you open tools and recover the last closed tool without using the mouse.

---

## Global shortcuts

| Shortcut | Action |
|----------|--------|
| **⌘K** (Mac) / **Ctrl+K** (Windows/Linux) | Open the **command palette** to search and open any tool by name, description, or category. |
| **Ctrl+Shift+T** | **Reopen last closed tool** (VS Code-style). Restores the tool you closed most recently. |

---

## Command palette

When the command palette is open:

- **↑ / ↓** — Navigate the list of matching tools.
- **Enter** — Open the selected tool.
- **Esc** — Close the palette without opening a tool.

The palette shows a hint: *↑↓ Navigate · Enter Open · Esc Close*. It may also show *Ctrl+Shift+T Reopen last tool*.

---

## Where shortcuts are shown

- **Sidebar search**: The search input placeholder mentions the command palette shortcut (e.g. “Search… or ⌘K”).
- **Tool search**: If a tool has a search that can be replaced by the palette, the placeholder may say “⌘K to open from anywhere”.
- **In-app**: Hover or focus elements that trigger the palette may show a tooltip with the shortcut.

---

## Platform notes

- On Mac, **⌘** is the Command key.
- On Windows and Linux, **Ctrl** is used (e.g. Ctrl+K, Ctrl+Shift+T).
- Shortcuts are global when the app is focused (web or Electron).
