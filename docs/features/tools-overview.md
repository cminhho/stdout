## Finding tools

- **Sidebar**: Tools are listed by group. Use the search box to filter by name or description.
- **Command palette**: Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux), then type the tool name.

See [Keyboard Shortcuts](keyboard-shortcuts.md) for more.
