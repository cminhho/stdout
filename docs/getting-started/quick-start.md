# Quick Start

Get started with stdout in about 5 minutes.

---

## 1. Open the app

- **Web**: Go to https://stdout-tools.web.app/
- **Desktop**: Launch stdout from your applications or Spotlight (macOS).

---

## 2. Find a tool

- Use the **sidebar** on the left: tools are grouped (Formatters, Encode and Crypto, Converters, etc.). Click a tool to open it.
- Or press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux) to open the **command palette**, then type the tool name (e.g. "JSON") and press Enter.

---

## 3. Use a tool (example: JSON Formatter)

1. Open **JSON Format/Validate** from the sidebar or command palette.
2. Paste or type JSON in the **Input** pane (e.g. `{"name":"stdout","tools":59}`).
3. The **Output** pane shows formatted (pretty-printed) JSON.
4. Use **Copy** or **Download** in the output toolbar to use the result elsewhere.

---

## 4. Try more

- **Base64**: Encode or decode text.
- **JWT Debugger**: Paste a JWT token to see header and payload.
- **Regex Tester**: Enter a pattern and test string to see matches.

Tools that support **sessions** (JSON Formatter, Base64, JWT Debugger) let you save and load named sessions from the page toolbar.

---

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘K / Ctrl+K | Open command palette (search and open any tool) |
| Ctrl+Shift+T | Reopen last closed tool |
